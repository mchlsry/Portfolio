About Me

Saya merupakan seorang mahasiswa sarjana tahun terakhir jurusan Computer Science dengan minat kuat di bidang Front-End Web Development. Saat ini, saya sedang menjalani program Studi Independen dengan topik Data Science & AI. Saya memiliki keterampilan dalam HTML, CSS, dan JavaScript. Saat ini saya masih mempelajari beberapa dasar framework yang nantinya dapat berguna dalam pembuatan suatu Web.

Aktif dalam organisasi kemahasiswaan, saya telah memegang beberapa peran penting dalam berbagai event, antara lain sebagai anggota divisi acara, hubungan masyarakat, dan sekretaris. Saya mampu bekerja di bawah tekanan, mengelola waktu dengan baik, dan selalu menyelesaikan tugas tepat waktu. Saya bersemangat untuk menerapkan pengetahuan dan keterampilan saya dalam pengembangan web untuk menciptakan pengalaman pengguna yang menarik dan responsif.
